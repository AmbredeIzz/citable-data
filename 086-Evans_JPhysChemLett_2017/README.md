- [`comsolinput/spherical_composite.mph`](comsolinput/spherical_composite.mph): COMSOL input files used for the simulation of the spherical composites
- [`comsolinput/MMM_50.mph`](comsolinput/MMM_50.mph): COMSOL input files used for the simulation of the mixed matrix membranes
- [`comsolinput/NLC.mph`](comsolinput/NLC.mph): COMSOL input files used for the simulation of composites with anisotropic adsorbent material
